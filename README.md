# CodeVault

A full-stack developer productivity platform for saving, organizing, searching, and managing reusable code snippets efficiently.

CodeVault helps developers store reusable code snippets with advanced tagging, multi-language support, secure authentication, and AI-assisted code suggestions to improve development workflow and productivity.

---

## 🚀 Features

- User Registration and Login
- JWT-Based Authentication
- Save and Manage Code Snippets
- Advanced Tagging System
- Multi-language Code Support
- Smart Search Functionality
- Favorite / Bookmark Snippets
- Snippet Categories and Filters
- AI-Assisted Code Suggestions
- RESTful API Integration
- Dockerized Deployment
- Responsive Modern UI

## 🔐 Authentication

CodeVault uses JWT (JSON Web Token) based authentication for secure login and user access management.

Features include:

- Secure Login & Signup
- Token-based Authorization
- Protected Routes
- Role-based Access (Future Scope)
